# WooCommerce Plugin Suite — Full Code Review

5 standalone WooCommerce plugins for Gumroad marketplace. Each is a single-file PHP plugin with settings API, meta boxes, and WooCommerce hooks.

All plugins use: singleton pattern, nonce verification, sanitize_text_field/sanitize_hex_color input handling, esc_html/esc_attr output escaping, current_user_can capability checks, register_uninstall_hook cleanup.

	// Delete all post meta.

		'post_type'      => array( 'product', 'shop_order' ),

	delete_option( 'wc_wholesale_settings' );

	$meta_keys = array(
		'_wc_wholesale_price',
		'_wc_wholesale_qty_rules',
		'_wc_wholesale_min_qty',
		'_wc_wholesale_disable',
	);

	$products = get_posts( array(
		'post_type'      => 'product',
		'posts_per_page' => -1,
		'fields'         => 'ids',
	) );

	foreach ( $products as $pid ) {
		foreach ( $meta_keys as $key ) {
			delete_post_meta( $pid, $key );
		}
	}
}
